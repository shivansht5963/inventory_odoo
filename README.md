## üìã Overview

Inventory Odoo is a comprehensive stock management solution built for franchise networks and individual vendors. The system provides two distinct access levels:

### Admin Dashboard
- **Full system access** to manage all operations
- Complete control over vendor accounts and permissions
- System-wide reporting and analytics
- Access with the demo credentials above

### Vendor Portal
- **Limited vendor access** for individual stock management
- Track inventory in real-time
- Manage deliveries and receipts
- Place and monitor orders
- Access with the demo credentials above
